# CARA PEMAKAIAN APLIKASI ASMARA BEDA FAKULTAS #
1.  Pastikan semua komputer yang akan dipakai berada pada jaringan yang sama
2.  Ubah variabel "IP_ADDRESS" pada const.py sesuai dengan IP Address dari server yang dipakai
3.  Jalankan "server.py"
4.  Jalankan "client-gui.py"
5.  Masukkan username yang unik
6.  Masukkan password sesuai dengan SERVER_PASS yang ada pada file const.py


# CARA MENGIRIM PESAN PADA APLIKASI ASMARA BEDA FAKULTAS #
1.  Tuliskan pesan secara langsung untuk mengirimkan pesan kepada semua klien yang terhubung dengan server
2.  Tuliskan pesan dengan cara "<IP Address>-<Port>-<Pesan>" sesuai dengan informasi dari klien tujuan
    untuk mengirimkan sebuah private message
